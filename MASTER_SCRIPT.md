# 動画制作マスター脚本 ＆ Veoプロンプト集

## 1. ナレーション設定
- **キャラクター:** 30代後半〜40代、落ち着いたアルト（知性と包容力の「導き手」）
- **BGM:** 序盤は重厚なアンビエント、中盤から Azure Mist を想起させる透明感のあるピアノへ

## 2. 統合脚本案
### 【導入：砂利道との対峙】
- **映像:** 左右分割画面。左は「美しい朝食」、右は「積み上がった伝票と油まみれの機械」。
- **ナレーション:** 「日常の豊かさと、現場の厳しさ。一見、遠く離れたこの二つは、実は同じ『根』で繋がっています。歩きにくいその道を、ただ耐えるだけの時間に、終止符を打ちませんか？」

### 【核心：組織の神経系と実装】
- **映像:** 右画面の「砂利道」の上に、青白い光の網が広がっていく。
- **ナレーション:** 「Studio S.O が提供するのは、組織の『神経系』を構築すること。現場の『砂利』を、経営を支える『ステッキ』へとローカル翻訳します。」
